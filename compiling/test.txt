00 -- Program / Sys Commands
############################

0001 | JMP <n>                  # Jumps to the <n>th line in the Program if r1 = 0


01 -- Data Transfer & Management
################################

0000 | PUT <n> r1
1000 | CPY <rA> <rB>

0100 | LOD <address> r1
1100 | WRD r1 <address>

0010 | LOR <address> r1
1010 | WRR r1 <address>

10 -- ALU
#########

0000 | ADD <rA> <rB>
0001 | SUB <rA> <rB>
0010 | INC <rA>
0011 | DEC <rA>

0100 | AND <rA> <rB>
0101 | ORR <rA> <rB>
0110 | XOR <rA> <rB>
0111 | NOT <rA> 

Ticks:

0 -> load instruction; cleanup
1 -> [load value 1]
2 -> [load value 2]
3 -> [save value]; next instruction
